# Coffee-Shop-Sales-Dashboard
Dashboard analisis penjualan berbasis data yang mengeksplorasi performa bisnis coffee shop berdasarkan waktu, lokasi toko, dan produk. Project ini dibangun menggunakan Microsoft Excel (Pivot Table &amp; Dashboard) untuk menunjukkan kemampuan data cleaning, analisis bisnis, visualisasi data, serta penyusunan insight dan rekomendasi berbasis data.
